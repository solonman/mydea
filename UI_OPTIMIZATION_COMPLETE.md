## 💡 最佳实践

### 使用建议
1. **保持一致性**：所有新组件都应使用 theme.css 中的变量
2. **复用样式类**：优先使用 `.card-glass`, `.btn-primary` 等
3. **渐进增强**：确保基础功能在旧浏览器也可用
4. **性能优先**：避免过度使用 blur 和 shadow

### 维护指南
1. **颜色修改**：只需更新 CSS 变量
2. **新增组件**：参考现有组件风格
3. **动画调整**：使用统一的 duration 和 easing
4. **测试覆盖**：检查 Chrome, Safari, Firefox
